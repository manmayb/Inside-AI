# Project summary (index)

Full product documentation lives in **[docs/PRODUCT/](./docs/PRODUCT/)** and **[docs/UX/](./docs/UX/)**.

## One sentence

**Inside AI** is a calm, chapter-based browser journey that simulates how a language model processes your message — designed for beginners, with optional technical depth.

## User path

1. Welcome — “See how AI thinks.”  
2. Prelude — three ideas before simulation  
3. Fifteen full-screen chapters  
4. Finale — summary, replay, or new question  

## What we are not

Not a real LLM client, observability platform, or research benchmark.
